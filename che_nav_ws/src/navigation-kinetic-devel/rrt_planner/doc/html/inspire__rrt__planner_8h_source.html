186  * 求两个向量之间的旋转角
187  * 首先明确几个数学概念：
188  * 1. 极轴沿逆时针转动的方向是正方向
189  * 2. 两个向量之间的夹角theta， 是指(A^B)/(|A|*|B|) = cos(theta)，0<=theta<=180 度， 而且没有方向之分
190  * 3. 两个向量的旋转角，是指从向量p1开始，逆时针旋转，转到向量p2时，所转过的角度， 范围是 0 ~ 360度
191  * 计算向量p1到p2的旋转角，算法如下：
192  * 首先通过点乘和arccosine的得到两个向量之间的夹角
193  * 然后判断通过差乘来判断两个向量之间的位置关系
194  * 如果p2在p1的顺时针方向, 返回arccose的角度值, 范围0 ~ 180.0(根据右手定理,可以构成正的面积)
195  * 否则返回 360.0 - arecose的值, 返回180到360(根据右手定理,面积为负)
196  */

199  const double epsilon = 1.0e-6;
200  const double nyPI = acos(-1.0);
201  double dist, dot, degree, angle;
202 
203  // normalize
204  dist = sqrt( x1 * x1 + y1 * y1 );
205  x1 /= dist;
206  y1 /= dist;
207  dist = sqrt( x2 * x2 + y2 * y2 );
208  x2 /= dist;
209  y2 /= dist;
210  // dot product
211  dot = x1 * x2 + y1 * y2;
212  if ( fabs(dot-1.0) <= epsilon )
213  angle = 0.0;
214  else if ( fabs(dot+1.0) <= epsilon )
215  angle = nyPI;

223  // vector p2 is clockwise from vector p1
224  // with respect to the origin (0.0)
225  if (cross < 0 )
